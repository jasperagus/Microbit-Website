Waarom space invaders?



Space invaders is een classic. ik heb het gekozen omdat het een goeide starter is plus je kan veel uitbreiden bij deze game.
de base is een player die op een enemy schiet, maar je kan power-ups, bosses, endings adden.

Voor mijn game heb ik de Microbit gebruikt. De microbit is een kleine computer die gemaakt is voor de "first steps" in programeren.
In de microbit kan je ook games maken. natuurlijk geen Cyberpunk of Need for speed, maar een simpelen space invaders wel.
